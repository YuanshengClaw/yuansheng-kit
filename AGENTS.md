# README for Agents

## Commit Messages

- Commit messages must follow the Conventional Commits specification.
- Limit every line to 80 characters.
- Limit the subject line to 72 characters.

## Skills

When working on this repository, use these skills by language:

- Nix (`.nix`): `nix-coding`
- Python (`.py`): `python-coding`
- TypeScript (`.ts`): `typescript-coding`
